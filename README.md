## Project Structure

```text
portfolio/
│
├── images/
│
├── style.css
│
├── script.js
│
├── index.html
|
└── README.md
```

| Folder/File | Purpose                                          |
| ----------- | ------------------------------------------------ |
| assets/     | Stores images of Projects                        |
| style.css   | Contains styling files                           |
| script.js   | Contains JavaScript functionality                |
| index.html  | Main entry point of the website                  |
| README.md   | Project documentation                            |
